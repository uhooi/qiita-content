## 「OWASP」とは？

「Open Web Application Security Project」の略で、Webアプリをはじめとするセキュリティ環境の現状や、セキュアなアプリ開発を促進する技術やプロセスに関する情報共有と普及を目的としたOSSコミュニティです。
